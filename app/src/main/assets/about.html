Welcome!

This is wATL library demo application, shows some features of TextViewEx - drop-in replacement for standart android.view.TextView with some additional features:

  • full text justification
  • image wrap
  • hyphenation supports
  • supports animation DynamicDrawableSpan
  • experimental support for bidirectional text
  • pagination adapter
